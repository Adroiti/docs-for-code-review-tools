Pattern: Missing plural rule

Issue: -

## Description

Enforces certain plural rules to always be specified/forbidden in a message.

```json
enum LDML {
  zero = 'zero',
  one = 'one',
  two = 'two',
  few = 'few',
  many = 'many',
  other = 'other',
}
```

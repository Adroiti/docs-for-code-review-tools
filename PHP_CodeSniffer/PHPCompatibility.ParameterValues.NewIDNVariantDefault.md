Pattern: Use of implicit `$variant` parameter

Issue: -

## Description

The default value for the `$variant` parameter has changed from `INTL_IDNA_VARIANT_2003` to `INTL_IDNA_VARIANT_UTS46` in PHP 7.4.

PHP version 7.4

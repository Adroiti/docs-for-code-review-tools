Pattern: Use of _PHP 4_ constructor

Issue: -

## Description

Bans PHP 4 style constructors.

Favor PHP 5 constructor syntax, which uses `function __construct()`. Avoid PHP 4 constructor syntax, which uses `function ClassName()`.

Pattern: Use of deprecated _MbString_ modifiers

Issue: -

## Description

Checks for use of deprecated and removed regex modifiers for _MbString_ regex functions.

Initially just checks for the PHP 7.1 deprecated `e` modifier.

PHP version 7.1

Pattern: Assigning the return value of `new` by reference

Issue: -

## Description

Detects the use of assigning the return value of `new` by reference.

This syntax has been deprecated since PHP 5.3 and removed in PHP 7.0.

PHP version 5.3
PHP version 7.0

Pattern: Missing whitespace around operator

Issue: -

## Description

There should be one space before and after all operators.

### Example of **incorrect** code:

```python
if age>15:
    print('Can drive')
```

### Example of **correct** code:

```python
if age > 15:
    print('Can drive')
```

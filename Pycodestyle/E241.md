Pattern: Use of multiple spaces after `,`

Issue: -

## Description

There should be only one space after the  `,` character.

### Example of **incorrect** code:

In this example there are two spaces after the comma

```python
my_tuple = 1,  2
```

### Example of **correct** code:

```python
my_tuple = 1, 2
```

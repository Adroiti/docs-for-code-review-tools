Pattern: Missing whitespace after `,`/`;`/`:`

Issue: -

## Description

There should be whitespace after the characters `,`, `;`, and `:`.

### Example of **incorrect** code:

```python
my_tuple = 1,2,3
```

### Example of **correct** code:

```python
my_tuple = 1, 2, 3
```

Pattern: Missing whitespace around arithmetic operator

Issue: -

## Description

There should be one space before and after an arithmetic operator (`+`, `-`, `/`, and `*`).

### Example of **incorrect** code:

```python
age = 10+15
```

### Example of **correct** code:

```python
age = 10 + 15
```

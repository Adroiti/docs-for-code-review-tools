Pattern: Missing whitespace around `%`

Issue: -

## Description

The modulo operator (`%`) should have whitespace on either side of it.

### Example of **incorrect** code:

```python
remainder = 10%2
```

### Example of **correct** code:

```python
remainder = 10 % 2
```

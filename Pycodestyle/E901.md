Pattern: SyntaxError or IndentationError

Issue: -

## Description

The code is not in valid Python syntax.

### Example of **incorrect** code:

This code is not valid Python.

```python
if (this.isNotPython()) {
    # Syntax error!
}
```

### Example of **correct** code:

```python
if self.isPython():
    # Happily chug along
```

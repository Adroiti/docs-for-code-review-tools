Pattern: Use of multiple spaces after operator

Issue: -

## Description

There should be only one space after an operator.

### Example of **incorrect** code:

```python
# There are two spaces after the multiplication operator
num = 10
doubled = num *  2
```

### Example of **correct** code:

```python
num = 10
doubled = num * 2
```

Pattern: Use of multiple spaces before operator

Issue: -

## Description

There should be only one space before an operator.

### Example of **incorrect** code:

```python
# There are two spaces before the multiplication operator
num = 10
doubled = num  * 2
```

### Example of **correct** code:

```python
num = 10
doubled = num * 2
```

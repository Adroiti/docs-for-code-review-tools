Pattern: Use of tab after `,`

Issue: -

## Description

There should be only one space after the `,` character.

### Example of **incorrect** code:

**Note:** The `→` character represents a tab.

```python
my_tuple = 1,→2,→3
```

### Example of **correct** code:

```python
my_tuple = 1, 2, 3
```

Pattern: Unnecessary `;`

Issue: -

## Description

Unlike programming languages like Java and C#, statements in Python do not need to end with a semicolon. They can be removed.

### Example of **incorrect** code:

```python
print('Hello world!');
```

### Example of **correct** code:

```python
print('Hello world!')
```

Pattern: Inconsistent class check

Issue: -

## Description

This rule enforces consistent use of `Object#is_a?` or `Object#kind_of?`.

## Default configuration

Attribute | Value
--- | ---
EnforcedStyle | is_a?
SupportedStyles | is_a?, kind_of?

Pattern: Empty `when`

Issue: -

## Description

This rule checks for the presence of `when` branches without a body.

## Examples

```ruby
# bad

case foo
when bar then 1
when baz then # nothing
end
```
```ruby
# good

case foo
when bar then 1
when baz then 2
end
```

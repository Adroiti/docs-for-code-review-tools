Pattern: Class has too many lines

Issue: -

## Description

This rule checks if the length a class exceeds some maximum value.
Comment lines can optionally be ignored.
The maximum allowed length is configurable.

## Default configuration

Attribute | Value
--- | ---
CountComments | false
Max | 100

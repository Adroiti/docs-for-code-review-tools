Pattern: Line is too long

Issue: -

## Description

This rule checks the length of lines in the source code. The maximum length is configurable.

## Default configuration

Attribute | Value
--- | ---
Max | 120
AllowHeredoc | true
AllowURI | true
URISchemes | http, https
IgnoreCopDirectives | false

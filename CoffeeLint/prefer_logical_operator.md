Pattern: Missing use of logical operator

Issue: -

## Description

This rule prohibits `is`, `isnt`, `not`, `and`, `or`, `yes`, `on`, `no`, `off`. Use `==`,
`!=`, `!`, `&&`, `||`, `true`, `false` instead.
